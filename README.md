📱 About The App

DayTask is a clean and intuitive task management mobile application built with Flutter. It helps users organize their daily tasks, track progress, schedule work, and collaborate with team members — all in a beautiful dark-themed UI powered by Cubit state management.

✨ Features

🏠 Home — Overview of completed and ongoing projects with progress tracking
📋 Task Details — Full task info, project progress, subtasks, due date and team members
➕ Add Task — Create new tasks with title, details, team members, time and date
📅 Schedule / Calendar — View today's tasks organized by date and time
💬 Messages — Chat and group messaging with team members
📲 Bottom Navigation Bar — Smooth navigation between all main screens
🌙 Dark Theme — Beautiful modern dark UI throughout the app

🛠️ Tech Stack

TechnologyUsageFlutter & DartUI & App LogicCubit (BLoC)State ManagementMaterial DesignUI Components
